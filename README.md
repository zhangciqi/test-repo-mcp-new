# 2048 游戏 - 最终版

基于 HTML5 的经典 2048 游戏，支持网格切换、主题换肤、成就系统、排行榜、分享功能等完整特性。

## 在线地址

https://zhangciqi.github.io/test-repo-mcp-new/

## 功能清单

### 核心玩法
- 支持 3×3 / 4×4 / 5×5 网格切换
- 方向键控制方块移动，相同数字碰撞合并
- 目标合成 2048，通关后可继续挑战更高分数

### 交互体验
- P键暂停/继续游戏，暂停遮罩覆盖
- 触摸滑动支持（移动端适配）
- 启动加载动画（Logo渐入+方块弹跳）
- 开局/通关/失败过场动画
- 音效反馈（可选）

### 数据持久化
- 本地存储最高分、步数、游戏时长
- 排行榜记录（前10名，支持昵称、按分数/时间排序）
- 主题偏好保存（亮色/暗色）
- 游戏设置保存（网格、速度、音效、动画）

### 成就系统
| 成就 | 名称 | 解锁条件 |
|------|------|----------|
| ACH_001 | 首次通关 | 合成 2048 方块 |
| ACH_002 | 满分挑战 | 达到 4096 或更高 |
| ACH_003 | 速度之王 | 60 秒内合成 1024 |

### 分享功能
- 复制战绩：一键复制分数、步数、时长文本到剪贴板
- 导出截图：Canvas 绘制游戏棋盘并下载 PNG 图片

### 其他功能
- 个人简历页：展示昵称、总游戏次数、最高分、通关次数、成就列表
- 新手引导：首次打开弹出操作指引，支持"不再提示"
- 设置面板：网格大小、游戏速度、音效、动画开关

## 技术栈

- **HTML5** - 页面结构与 Canvas 截图
- **CSS3** - 渐变、动画、CSS Grid 布局、响应式
- **JavaScript ES6+** - 游戏逻辑与模块化架构
- **LocalStorage** - 数据持久化

## 本地运行

```bash
# 克隆项目
git clone https://github.com/zhangciqi/test-repo-mcp-new.git

# 进入项目目录
cd test-repo-mcp-new

# 使用 Python 简易服务器
python -m http.server 8000

# 或直接用浏览器打开 index.html
```

## GitHub Pages 部署

1. 进入项目仓库 → Settings → Pages
2. Source 选择 `main` 分支，路径选择 `/ (root)`
3. 点击 Save，等待自动构建
4. 访问 `https://zhangciqi.github.io/test-repo-mcp-new/`

## 操作说明

### 键盘操作
- `↑` `↓` `←` `→` - 移动方块
- `P` / `p` - 暂停/继续游戏

### 鼠标操作
- 新游戏 - 重新开始
- 暂停 - 暂停游戏
- 设置 - 调整游戏参数
- 排行榜 - 查看历史记录（支持排序/清空）
- 主题 - 切换明暗主题
- 简历 - 查看个人游戏数据统计

### 触屏操作
- 滑动屏幕 - 移动方块
- 点击按钮 - 执行对应操作
