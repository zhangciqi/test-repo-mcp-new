# 2048 游戏

一个使用 HTML、CSS 和 JavaScript 开发的经典 2048 益智游戏。

## 项目概述

这是一个完整的 2048 网页游戏，实现了经典游戏的所有核心功能，并支持桌面端和移动端。

## 功能特性

### P0 功能（核心功能）✅

- ✅ 4x4 网格渲染
- ✅ 方块生成（90% 概率为 2，10% 概率为 4）
- ✅ 方块移动（上下左右）
- ✅ 方块合并（相同数字合并）
- ✅ 分数计算（每次合并得分等于合并后方块的数字）
- ✅ 胜利判定（合成 2048）
- ✅ 失败判定（无法移动）
- ✅ 新游戏功能

### P1 功能（用户体验）✅

- ✅ 移动动画（150ms ease-in-out）
- ✅ 合并动画（缩放放大后恢复）
- ✅ 分数动画（跳动效果）
- ✅ 响应式布局（桌面端 + 移动端）
- ✅ 方块颜色渐变（符合官方颜色规格）

### P2 功能（可选）✅

- ✅ 本地最高分存储（使用 localStorage）

## 技术栈

- **HTML5**: 语义化结构
- **CSS3**: 响应式设计，动画效果
- **JavaScript**: ES6+ 面向对象编程
- **localStorage**: 本地数据持久化

## 游戏操作

### 桌面端

- 使用方向键 ↑ ↓ ← → 移动方块

### 移动端

- 在游戏区域滑动屏幕移动方块

## 游戏规则

1. 通过移动所有方块，使相同数字的方块合并
2. 每次移动后，会在随机空格中生成一个新的方块（2 或 4）
3. 合并得分 = 合并后方块的数字
4. 当合成 2048 方块时获胜
5. 当无法移动且网格满时游戏失败
