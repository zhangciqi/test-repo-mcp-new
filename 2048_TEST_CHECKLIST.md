## 测试技巧

### 如何模拟移动端触摸操作
1. 打开Chrome开发者工具（F12）
2. 点击左上角的"切换设备模拟"图标（或按Ctrl+Shift+M）
3. 选择移动设备（如iPhone、iPad）
4. 使用鼠标模拟触摸滑动

### 如何查看localStorage
1. 打开Chrome开发者工具（F12）
2. 切换到"Application"标签
3. 在左侧菜单中选择"Local Storage" > "file://"
4. 查看key: "bestScore2048"

### 如何测试游戏结束条件
1. 故意制造一个无法继续的局面
2. 例如：让网格填满且没有相邻相同数字
3. 观察失败遮罩是否正确弹出

## 快速测试脚本（浏览器Console）

在浏览器Console中执行以下代码进行快速测试：

```javascript
// 检查游戏是否初始化
console.log("检查游戏初始化...");
const game = window.game2048 || document.querySelector('.tile');
console.log("游戏元素存在:", !!game);

// 检查分数显示
const score = document.getElementById('current-score');
console.log("分数元素存在:", !!score);
console.log("当前分数:", score.textContent);

// 检查最高分
const best = document.getElementById('best-score');
console.log("最高分元素存在:", !!best);
console.log("最高分:", best.textContent);

// 检查localStorage
const savedScore = localStorage.getItem('bestScore2048');
console.log("保存的最高分:", savedScore);

// 统计方块数量
const tiles = document.querySelectorAll('.tile');
console.log("当前方块数量:", tiles.length);
console.log("方块数值:", Array.from(tiles).map(t => t.textContent));

console.log("快速测试完成！");
```
