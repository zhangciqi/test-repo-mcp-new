## 🎯 使用说明

### 快速开始
1. 克隆或下载代码
2. 使用本地服务器打开 `index.html`
   ```bash
   # Python方式
   python -m http.server 8080
   # 访问 http://localhost:8080

   # Node.js方式
   npx serve .
   ```

### 游戏操作
- **桌面端**: 使用方向键 ↑ ↓ ← → 移动方块
- **移动端**: 滑动屏幕移动方块
- **新游戏**: 点击"新游戏"按钮重新开始

### 游戏规则
1. 使用方向键移动所有方块
2. 相同数字的方块会合并成一个
3. 每次移动后生成一个新方块（2或4）
4. 目标：合成2048方块
5. 当无法移动时游戏结束
